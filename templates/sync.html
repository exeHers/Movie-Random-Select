How syncing works

  • Shared mode — the app is connected to PostgreSQL (DATABASE_URL). All devices use the same data.
  • Local mode — no DATABASE_URL: data stays in a file on the machine running the server (fine for your laptop only).

Host it (simple path)

  1. Create a free Postgres DB (e.g. Neon or Supabase).
  2. Copy the connection string as DATABASE_URL (include ?sslmode=require if the provider shows it).
  3. Deploy this app (e.g. Render, Fly.io, any VPS) with TMDB_API_KEY and DATABASE_URL set.
  4. Set PUBLIC_BASE_URL to your live https:// URL (no trailing slash) so “Copy app link” is correct.

In the project repo: TUTORIAL.md (local run, deploy, Android APK) and DEPLOY-FREE.md (free Postgres + hosting).
