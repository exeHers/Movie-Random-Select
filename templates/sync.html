How syncing works

  • Shared mode — the app is connected to PostgreSQL (DATABASE_URL). All devices use the same data.
  • Local mode — no DATABASE_URL: data stays in a file on the machine running the server (fine for your laptop only).

Host it (simple path)

  1. Create a free Postgres DB (e.g. Neon, Supabase, or Railway).
  2. Copy the connection string as DATABASE_URL (use ?sslmode=require if the host asks).
  3. Deploy this app (Fly.io, Railway, Render, a small VPS) with TMDB_API_KEY and DATABASE_URL set.
  4. Set PUBLIC_BASE_URL to your live URL so “Copy app link” always matches.
